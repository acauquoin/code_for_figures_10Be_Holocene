# Python code for the figures in the article on beryllium-10 data over the last seven millennia, by Jouzel et al.

## Reference of the paper
Jouzel, J., Cauquoin, A., Bard, E., Zhang, L., Hou, S., Wu, Z., Zhou, W., Lipenkov, V., Petit, J.-R., Raisbeck, G., and Yiou, F.: Beryllium 10 in Antarctica over the last seven millennia, <i>Sci. Data</i>, in review.

## Details of the GitHub repository
The python code and the necessary input files are provided for each figure in the folders <b>figure*</b>. 

The figures are compiled in the folder <b>compilation_figures</b>.

## Where to download the data
The data can be found in the Figshare database at https://doi.org/10.6084/m9.figshare.29623730.
